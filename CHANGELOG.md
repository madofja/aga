# Changelog

## [1.0.0] - 2026-03-31

### Added
- Initial release of **Whisper Network: Sentient Society**.
- Full simulation gameplay with procedural terrain, dynamic weather, autonomous agents, language mutation, and social economics.
- Runtime observer dashboard and controls.
- Local executable build scripts for Windows and Linux/macOS.
- Automated GitHub Actions release workflow that builds binaries and publishes tagged releases (`v*`).
